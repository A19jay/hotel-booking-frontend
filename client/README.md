# 🏨 Hotel Booking Frontend

A modern and responsive **Hotel Booking Frontend** built with **React.js**, **Vite**, and **Tailwind CSS**. The application provides an elegant user interface for exploring hotels, viewing room details, and experiencing a seamless booking flow.

---

## ✨ Features

- 🏠 Beautiful Landing Page
- 🏨 Browse Available Hotels & Rooms
- 🔍 Search and Filter Interface
- 📄 Room Details Page
- 📅 Booking UI
- 👤 My Bookings Page
- 🏢 Hotel Owner Dashboard
- 📱 Fully Responsive Design
- ⚡ Fast Navigation with React Router
- 🎨 Modern UI with Tailwind CSS

---

## 🛠️ Tech Stack

| Technology        | Purpose                  |
| ----------------- | ------------------------ |
| React.js          | Frontend Library         |
| Vite              | Development & Build Tool |
| Tailwind CSS      | Styling                  |
| React Router DOM  | Client-side Routing      |
| JavaScript (ES6+) | Application Logic        |

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/A19jay/hotel-booking-frontend.git
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
